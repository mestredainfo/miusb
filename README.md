# MIUSB

O MIUSB é um software desenvolvido para remover dispositivos portáteis, como pendrives e HDs externos, com máxima segurança.

## Requerimento

- Sistema Operacional: Linux Debian 12 ou superior, Linux Ubuntu 22.04 ou superior
- Arquitetura: x64
